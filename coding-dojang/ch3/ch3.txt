SyntaxError: invalid syntax: 구문 에러로 보통 다음과 같은 이유로 발생한다 
1.괄호, 따옴표, 콜론, 세미콜론 등의 문법적인 구문이 잘못됐을 때
2.들여쓰기가 잘못됐을 때
3.정의되지 않은 변수나 함수를 사용했을 때
4.함수나 메소드 등의 호출 방식이 잘못됐을 때

코드를 한 줄 한 줄 실행하여 결과를 얻는 방식을 인터프리터(interpreter) 방식이라고 부릅니다.

파이썬 코드를 직접 입력해서 실행하는 프로그램을 파이썬 셸(Python Shell)이라고 하며 >>> 부분을 파이썬 프롬프트(Python prompt)라고 부릅니다.
파이써 쉘에 코드 입력하면 인터프리티터가 해석해 준다

프로그래밍에서 코드를 파일 형태로 저장한 것을 소스 코드 또는 소스 파일이라고 부릅니다.

q.소스파일과 스크립트 파일의 차이점은?
a.둘다 코드 담은 파일인데 소스파일은 컴파일러를 사용해서 실행 파일로 변환해야 사용 가능 하고(ex.java) 스크립트 파일은 한 줄씩 읽어서 실행하는 파일임(ex.python)

cmd 여는 방법
1.win +R (파일 탐색기를 연다)
2.cmd를 검색한다.
(그냥 메뉴에서 cmd검색하면 되는 거 아닌가 싶음)

하지만 나는 터미널을 사용하니 조금 다르게 한다
1.win+x
2 터미널 열기

터미널을 연 후 pyhton을 입력하면 python의 기능을 사용할 수 있다.
명령 프롬프트에서 파이썬 셸을 끝내려면 exit()를 입력하거나 Ctrl+Z를 누릅니다.

Q.cmd에서 pyhton을 입력하면 python을 설치하라 하고 터미널에서는 pyhton이 실행되는 걸까
A.cmd는 path에 등록을 안하면 못 찾는다고 한다

터미널에서 스크립트 파일이 있는 경로에서 python 스크립트파일 이름.py 을 입력하면 스크립트 파일 실행 가능 (python 스크립트파일 이름.ipynb 는 안됨)

print ('Hello, world!' )
